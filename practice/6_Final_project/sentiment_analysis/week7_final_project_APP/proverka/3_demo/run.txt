1. Открываем папку как проект в Pycharm (интерпретатор питон 2.7, это видно preferences -> project interpreter)
2. Запускаем файл demo
3. Открываем ссылку 0.0.0.0:2525/sentiment-demo
4. Пробуем (вводим отзывы на английском)

Если вдруг выпадают ошибки, убедитесь что sklearn, scipy и другие бибилиотеки имеют последние версию в установленном интерперататаре.
Все это видно в preferences -> project interpreter
